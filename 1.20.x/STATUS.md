# World Height Limit 1.20.x

## Status

Narrowed support branch.

## Notes

- Separate project exists for `1.20.5` and `1.20.6`
- `1.20.0` through `1.20.4` are not supported in this branch
- Earlier `1.20.x` builds use different `dimension_type` datapack fields and currently break startup
- Use this branch only for `1.20.5-1.20.6` testing
