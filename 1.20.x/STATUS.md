# World Height Limit 1.20.x

## Status

Smoke-tested and working.

## Notes

- Separate project exists for `1.20.5` and `1.20.6`
- Test platform set: Bukkit, Spigot, Paper, Purpur, and Folia
- Verified on `1.20.6`: startup, config generation, datapack generation, and high-height building work
- Spigot/Bukkit port compiles against Spigot API and no longer requires Paper Adventure APIs
- `1.20.0` through `1.20.4` are not supported in this branch
- Earlier `1.20.x` builds use different `dimension_type` datapack fields and currently break startup
- Use this branch only for `1.20.5-1.20.6`
